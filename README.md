# She Supply Donation Tracker
A responsive React.js frontend web application designed to help nonprofits manage and track donation items. This demo application allows staff to add, categorize, and remove donation needs with a clean, modern interface.
This application demonstrates a real-world use case for a nonprofit organization like She Supply, providing an intuitive interface for managing donation inventory.
* https://img.shields.io/badge/React-18.2.0-blue
* https://img.shields.io/badge/Tailwind-CSS-38B2AC
* https://img.shields.io/badge/License-MIT-green

# Project Overview
* Add Donation Items: Input new items with name and category
* Categorize Items: Organize items into categories (Hygiene, Food, Clothing, Baby Care)
* Remove Items: Delete items when they are no longer needed
* Responsive Design: Works seamlessly on desktop and mobile devices
* Real-time Updates: Instant UI updates when modifying the donation list

View Live Application: https://she-supply-donation-tracker.netlify.app

# Features
* Interactive UI: Add and remove donation items with immediate visual feedback
* Category Management: Organize items into predefined categories
* Form Validation: Prevents empty submissions
* Responsive Layout: Adapts to different screen sizes using Tailwind CSS
* State Management: Efficient React state handling for dynamic updates
* Modern Design: Clean, professional interface suitable for nonprofit use

# Usage
Add an Item:
* Enter the item name in the text field
* Select a category from the dropdown
* Click "Add Item" or press Enter
  
Remove an Item:
* Click the "Remove" button next to any item
*The item is immediately removed from the list

View Summary:
* See total item count and category diversity at the bottom

# Technologies Used
* Frontend Framework: React 18 with Hooks (useState)
* Styling: Tailwind CSS for responsive design
* Build Tool: Direct browser execution with Babel Standalone
* Deployment: Netlify
